Application web sur les pays d'Afrique

J'ai réalisé ce projet pour répondre au bésoin d'un client mais aussi valider les compétences demandées pour le titre professionnel développeur web et web mobile

L'objectif ? Réaliser un cahier des charges.
Concevoir une maquette.
Concevoir et mettre en place une base de données.
Développer la partie front-end et la partie back-end.
L'authentification des utilisateurs y est mais elle est réservée qu'à l'administrateur pour le moment.

CRUD
Dans ce projet, l'administrateur a la possiblité de créer, d'ajouter, de modifier ou de supprimer un pays. Mon client, étant novice dans les nouvelles technologies, a la possibilité de modifier facilement les informations en ligne grâce à une interface d'administration sur mesure que j'ai réalisé.

J'ai respecté les normes de sécurité RGPD et j'ai réalisé la mise en production. Ce projet simple mais fonctionnel à donné une entière satisfaction à mon client.

Le site est hebergé chez ionos.
Voici les technologies utilisées : HTML, CSS, EJS, NodeJS, ExpressJS, MySQL.
